# Entrenamiento Auditivo - Escalas Musicales

Aplicación web para el entrenamiento auditivo de escalas tonales y modales.

## Características

- ✅ Reproducción de audio sintético de 10 tipos de escalas diferentes
- ✅ Sistema de opciones múltiples (4 opciones por ejercicio)
- ✅ Retroalimentación inmediata (correcta/incorrecta)
- ✅ Estadísticas en tiempo real (correctas, incorrectas, precisión)
- ✅ Interfaz simple y clara para estudiantes
- ✅ Diseño responsive para móviles y tablets

## Escalas Incluidas

### Tonales
- Mayor
- Menor Natural
- Menor Armónica
- Menor Melódica

### Modales
- Dórico
- Frigio
- Lidio
- Mixolidio
- Eólico
- Locrio

## Cómo Usar

1. Abre `index.html` en tu navegador
2. Haz clic en "Reproducir Escala" para escuchar
3. Selecciona la escala que crees que es
4. Recibe retroalimentación inmediata
5. Haz clic en "Siguiente Ejercicio" para continuar

## Tecnología

- HTML5
- CSS3 (diseño moderno y responsive)
- JavaScript vanilla (Web Audio API para síntesis de audio)

## Notas Técnicas

La aplicación utiliza la Web Audio API para generar las escalas de forma sintética en tiempo real. No requiere archivos de audio externos, todo se genera mediante código.
